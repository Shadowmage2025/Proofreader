# Proofreader
中文小说文本校对工具

一个基于 Python 的中文小说文本自动校对工具，集成了拼写纠错和语法检查功能。

功能特性

• ✅ 智能拼写纠错：基于 pycorrector 的错别字检测与纠正

• ✅ 语法结构分析：使用 HanLP 进行依存句法分析，识别潜在语法问题

• ✅ 标点符号优化：自动将英文标点转换为中文标点

• ✅ 白名单保护：自定义词汇保护，防止特定词汇被误改

• ✅ 批量处理：支持整本小说的章节批量校对

• ✅ 详细报告：生成详细的校对报告，记录所有修改内容


环境要求

• Python 3.8 或更高版本

• Windows 系统（已优化兼容性）

安装步骤

1. 克隆或下载项目文件

将项目文件保存到本地目录。

2. 安装依赖包

打开命令提示符或 PowerShell，执行以下命令：
pip install pycorrector natsort hanlp


3. 准备待校对的小说文件

在项目根目录下创建小说文件夹，例如：

项目根目录/proofreader.py
项目根目录/斗破苍穹/
项目根目录/斗破苍穹/第1章_陨落的天才.txt
项目根目录/斗破苍穹/第2章_斗气大陆.txt
……

使用方法

基本使用

1. 运行程序：
python proofreader.py


2. 根据提示输入小说文件夹名称（如：斗破苍穹）

3. 程序会自动处理所有章节，并在同级目录生成 {书名}_校对结果 文件夹

输出结构

项目根目录/斗破苍穹_校对结果/
项目根目录/斗破苍穹_校对结果/第1章_陨落的天才.txt
项目根目录/斗破苍穹_校对结果/校对报告_第1章_陨落的天才.txt
项目根目录/斗破苍穹_校对结果/第2章_斗气大陆.txt
项目根目录/斗破苍穹_校对结果/校对报告_第2章_斗气大陆.txt
……

配置说明

白名单配置

在代码的 CUSTOM_WHITELIST 部分添加需要保护的词汇：
CUSTOM_WHITELIST = {
    "消炎": "萧炎",      # 防止"萧炎"被误改为"消炎"
    "魂殿": "魂殿",      # 保护专有名词
    "纳兰嫣然": "纳兰嫣然",
}


格式说明："错误写法": "正确写法"

功能模块详解

1. 拼写纠错模块

• 基于统计语言模型检测错别字

• 支持自定义混淆集

• 轻量模式运行，无需额外语言模型

2. 语法检查模块

• 使用 HanLP 依存分析器

• 自动分句处理，避免长文本内存溢出

• 检测语法结构异常

3. 文本清洗模块

• 自动移除重复的章节标题

• 过滤章节结束标记（如"本章完"）

• 智能标点符号转换


使用建议

1. 备份原文件：校对前请备份原始文件
2. 分批次处理：对于超长小说建议分卷处理
3. 人工复核：校对后建议人工检查重要章节
4. 白名单配置：针对特定小说配置专有名词白名单

常见问题

1. 依赖安装失败：尝试使用国内镜像源
   pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pycorrector natsort hanlp
   

2. 内存不足：程序已优化内存使用，如遇问题可减少单次处理章节数量

3. 模型下载慢：程序自动使用清华镜像，如遇网络问题可手动配置代理
